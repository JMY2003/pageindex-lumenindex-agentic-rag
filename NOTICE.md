# Notices And Attribution

LumenIndex is a production-oriented document retrieval and question-answering web system inspired by the open-source PageIndex architecture.

PageIndex attribution:

- Project: PageIndex

LumenIndex keeps the PageIndex-inspired ideas of hierarchical document indexing, reasoning-based retrieval, tight evidence page reads, and vectorless RAG. The current LumenIndex web application is implemented as a FastAPI application with its own persistence, authentication, user isolation, frontend, settings, context compaction, and deployment stack.

This notice is included to make the relationship clear for users searching for PageIndex, pageindex web app, PageIndex RAG, and vectorless document retrieval systems.
